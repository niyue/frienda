Frienda

Analyzer for friends data in kaixin001.com

DESCRIPTION:

Frienda is a program used to analyze and visualize your friends network within kaixin001.com.

FEATURES:

  1. Grab your friends data from kaixin001.com
  2. Visualize friends network by outputting an undirected graph, which tells you how your friends, and friends of friends, relating to each other
  3. Support png and svg formats
  4. Support to use friend's name or avartar as node in the diagram

SAMPLE OUTPUTS:

PNG format sample download: http://www.box.net/shared/o4nuvlrd9f

SVG format sample download: http://www.box.net/shared/rtngd348k0

REQUIREMENTS:

  • Only tested under Mac OS X 10.5.6

INSTALL and RUN:

  1. Install Java 5
  2. Install Graphviz, http://www.graphviz.org/
  3. Install JRuby, http://jruby.org
  4. Install rgl and hpricot gems, jruby -S gem install rgl hpricot
  5. Download Batik SVG library 1.7, http://xmlgraphics.apache.org/batik/download.cgi
  6. Unzip and copy Batik library, unzip batik-1.7.zip
  7. Create jar directory, mkdir FRIENDA_HOME/lib/jar
  8. Copy unzipped Batik library, cp -r batik-1.7/* FRIENDA_HOME/lib/jar/
  9. Change config.yml according to parameter description
  10. Run "jruby bin/frienda.rb"
